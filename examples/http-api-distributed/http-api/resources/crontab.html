Crontab Scheduler

{NAV}

Cron Schedule

Set a cron expression to start periodic scheduling. When the cron fires, the crontab service calls back to this http-api, which prints an info log to verify the callback succeeded.

Format: minute hour day-of-month month day-of-week. Examples: */1 * * * * (every 1 min), */30 * * * * * (every 30 min), 0 * * * * (hourly)
